# **App Name**: Forge

## Core Features:

- Idea Intake & Parsing: Structures raw user input into a validated JSON object containing core problems, target user profiles, and key assumptions using AI.
- Automated Risk Register: Identifies 4-6 critical execution and market risks to reveal hidden project friction using an AI analysis pipeline.
- Scenario Persona Tool: A reasoning tool that generates three distinct founder scenarios with unique constraints, blind spots, and success definitions.
- 30/60/90 Milestone Synthesis Tool: A reasoning tool that creates a phased execution roadmap, identifying specific testable assumptions and success signals.
- The Experiment Gate: A mandatory UI decision point where users must select one 'first bet' from three AI-generated experiments before proceeding.
- Resource Mapping Tool: A reasoning tool that matches the chosen experiment to specific high-level support categories to guide resource retrieval.
- Verified Resource Retrieval: Rule-based filtering of a static 40-item dataset to match AI-generated categories with real-world grants and tools.
- Session Persistence: Maintains project context, chosen paths, and state across browser reloads using local storage implementation.

## Style Guidelines:

- Background: Warm off-white (#FBFBFA) for a soft, focused environment.
- Surfaces: Pure white (#FFFFFF) for cards and panels to elevate active planning content.
- Primary Text: Soft charcoal (#1A1A17) for sharp, readable typography.
- Secondary Text: Muted slate-gray (#6B6A65) for captions, subtitles, and labels.
- Dividers: Ultra-subtle hairlines (#EBEAE4) for clean section separation.
- Action Color: Muted Sage Green (#344E41) for buttons, progress bars, and selected states.
- Risk Color: Terracotta Red (#A24836) reserved specifically for risk registration and uncertainty warnings.
- Headings use Lora serif font for an authoritative feel; body text uses Inter sans-serif for clarity.
- A calm, centered layout with a max-width of 680px and a vertical progress stepper to guide the pipeline.
- Smooth fade-in transitions between stages to reinforce the sequential nature of the reasoning process.
- Minimalist glyphs and functional emojis used in the resource panel to differentiate categories visually.
